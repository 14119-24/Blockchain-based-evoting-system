Before registering, ensure you meet ALL requirements:

| Requirement | Details | Required |
|-------------|---------|----------|

| **Party** | Member of registered political party | ✅ Yes |
| **Conduct** | Good moral character, no criminal record | ✅ Yes |
| **Registration Fee** | 1000 USD (non-refundable) | ✅ Yes |

## 🚀 Getting Started

### For New Candidates

1. **Visit**: candidate-register.html
2. **Select**: "Register" tab
3. **Fill**: Personal information (Step 1)
4. **Verify**: Eligibility requirements (Step 2)
5. **Submit**: Campaign platform & payment (Step 3)
6. **Confirm**: Check email for registration confirmation
7. **Wait**: Admin verification (24-48 hours)
8. **Login**: Use dashboard after verification

### For Registered Candidates

1. **Visit**: candidate-register.html
2. **Select**: "Login" tab
3. **Enter**: Email and password
4. **Access**: Your dashboard
5. **Monitor**: Campaign statistics
6. **Track**: Real-time votes

## 🔒 Security Tips

- ✅ Use strong password (8+ chars, mix of uppercase, numbers, symbols)
- ✅ Keep login credentials confidential

- ✅ Don't share your candidate ID
- ✅ Use secure, private internet connection

## ❓ Troubleshooting

### Registration Issues

| Problem | Solution |
|---------|----------|
| "Age must be 21+" | Change date of birth to 21+ years ago |
| "BSc degree required" | Check the checkbox to confirm degree |
| "Party selection required" | Select party from dropdown |
| "Good conduct required" | Check the conduct confirmation |

### Login Issues

| Problem | Solution |
|---------|----------|
| "Invalid credentials" | Verify email and password are correct |
| "Payment pending" | Complete 1000 USD registration fee |
| "Verification pending" | Wait for admin approval |
| "Session expired" | Login again |

### Dashboard Issues

| Problem | Solution |
|---------|----------|
| "Not authenticated" | Login to candidate account |
| "No data showing" | Wait for votes to be cast |
| "Statistics not updating" | Refresh page (F5) or wait for automatic update |
